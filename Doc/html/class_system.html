Detailed Description

Classe qui gere le deroulement du jeu.

Cette classe est le moteur du Tamagotchi, elle communique egalement avec toute la partie graphique. Elle gere les evenements aleatoires, la communication utilisateur (menus, connexion a la partie graphique), le temps (evolution des caracteristiques du Tamagotchi ainsi que la gestion des parties (creer, sauvegarder, supprimer etc ...).
